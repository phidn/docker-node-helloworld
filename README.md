# Run your Node.js app from a Docker Container

## Requirements
* docker >= 20.10.8
* nodejs >= 12.22.1
* npm >= v6.14.14
* yarn >= v1.21.11

## Notes
- npm install
- docker build -t hello-world .
- docker run -p 8000:3000 hello-world
